## Description

Network Plus is a web application designed to help users visualize and manage their personal and professional networks. Track interactions, explore connections, and gain insights into the strength and frequency of your relationships. The app combines intuitive design with powerful features to make network management simple and engaging.

## Features

Contact Upload: Import contacts easily via CSV or VCF files.

Interaction Logging: Track calls, messages, and meetings with weighted scoring based on recency and platform.

Network Visualization: Interactive graphs that reflect relationship strength through node sizing.

Search & Filter: Quickly locate contacts and filter by interaction type.

Responsive Design: Works seamlessly on desktop and mobile devices.

Secure Data Storage: Safely stores user data with modern authentication options.

## Technologies

Frontend: React, Tailwind CSS, React Force Graph

Backend: Node.js, Prisma, PostgreSQL

Deployment: Vercel

Other Tools & Libraries: Supabase (optional authentication & database), CSV/VCF parsing utilities
